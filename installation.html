Installation

Option 1: Clone this repository

To copy the repository for your own purposes (including to suggest edits):

  1. Clone fetchAZA to your local machine: With a command line usage of git, this looks like:

    git clone https://github.com/ocean-uhh/fetchAZA

    Alternatively, in your web browser, navigate to https://github.com/ocean-uhh/fetchAZA and click the green button “<> Code” for a dropdown menu of options. Choose, e.g. “Open with GitHub Desktop”. This requires the application “GitHub Desktop” (https://github.com/apps/desktop). See instructions here.

  2. Change to the parent directory of fetchAZA on your local machine

  3. Install the package locally

pip install -r requirements-dev.txt
pip install -e .

Option 2: Install with pip

pip install fetchAZA
